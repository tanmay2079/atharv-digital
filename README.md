# Atharv Digital Photo Studio

## Capturing Moments, Crafting Memories

Welcome to **Atharv Digital Photo Studio**, a comprehensive platform dedicated to premium photography and digital imaging services. Located in Bhogewadi, Maharashtra, we blend traditional expertise with modern digital craftsmanship to capture and preserve your most cherished moments.

This monorepo houses both our cutting-edge web application and a companion mobile application, ensuring a seamless experience across all devices.

## 🌟 Features

### Web Application

- **Stunning Portfolio Display**: Showcase high-quality photography work with an elegant and responsive design.
- **Detailed Service Offerings**: Clearly outline a wide range of services, including:
    - Wedding Photography
    - Passport Photos
    - Event Coverage
    - Digital Imaging Services
- **Interactive Contact & Inquiry Forms**: Easy-to-use forms for clients to reach out and book services.
- **Modern UI/UX**: Built with a focus on aesthetics and user experience, featuring smooth animations and a clean interface.
- **Location-Aware Content**: Highlights services specific to Bhogewadi, Maharashtra.

### Mobile Application

- **Cross-Platform Accessibility**: A native-like experience on both iOS and Android devices.
- **Seamless Integration**: Complements the web application, providing on-the-go access to services and portfolio.
- **User-Friendly Interface**: Intuitive design for easy navigation and interaction.

## 🚀 Technologies Used

### Overall Monorepo Structure

- **Yarn Workspaces**: Efficient management of multiple projects within a single repository.
- **TypeScript**: Enhanced code quality and developer experience with static typing.

### Web Application (`apps/web`)

- **Next.js**: A powerful React framework for building fast, scalable, and SEO-friendly web applications.
- **React**: Declarative JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
- **Drizzle ORM**: A modern TypeScript ORM for type-safe database interactions.
- **Database**: SQLite for development, PostgreSQL for production.

### Mobile Application (`apps/mobile`)

- **Expo**: A framework and platform for universal React applications, enabling fast development of iOS, Android, and web apps.
- **React Native**: For building native mobile apps using React.
- **Tailwind CSS**: Consistent styling across web and mobile platforms.

## 🛠️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- Yarn (v1.22 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tanmay2079/atharv-digital.git
   cd atharv-digital
   ```
2. Install dependencies for the monorepo:
   ```bash
   yarn install
   ```

### Running the Web Application

1. Navigate to the web app directory:
   ```bash
   cd apps/web
   ```
2. Start the development server:
   ```bash
   yarn dev
   ```
   The web application will be accessible at `http://localhost:3000` (or another port if 3000 is in use).

### Running the Mobile Application

1. Navigate to the mobile app directory:
   ```bash
   cd apps/mobile
   ```
2. Start the Expo development server:
   ```bash
   yarn start
   ```
   Follow the instructions in your terminal to open the app on an emulator/simulator or your physical device using the Expo Go app.
